# Apache Hop
* https://hop.apache.org/
* https://github.com/apache/hop/

The Hop Orchestration Platform/编排平台, or Apache Hop, aims to facilitate all aspects of data and metadata orchestration.

Hop is an entirely new open source data integration platform that is easy to use, fast and flexible

Hop aims to be the future of data integration. 
- Visual development/可视化开发 enables developers to be more productive than they can be through code. 
- Our Design once, run anywhere workflows and pipelines/设计一次到处运行的工作流和流水线 can be designed in the Hop Gui and run on the Hop native engine (local or remote), or on Spark, Flink, Google Dataflow or AWS EMR through Beam. 
- Lifecycle Management/生命周期管理 enables developers and administrators to switch between projects, environments and purposes without leaving your train of thought.
